💜 Fintech — Tela de Dashboard

Dashboard de um projeto Fintech feito com HTML, CSS e Tailwind CSS.
Exibe saldo, receitas e despesas, evolução financeira, registros e metas.


✨ Sobre a tela


Header com a marca e o usuário (User123_)
Meu Saldo — saldo atual em destaque
Receitas e despesas — barra comparativa
Evolução Financeira — gráfico de linhas em SVG (6 meses)
Meus Registros — lista de lançamentos com legenda
Minhas metas — cadastro de metas



Sem JavaScript. Dados mockados direto no HTML.

📁 Estrutura

├── index.html => Estrutura e conteúdo da tela

├── styles.css  => Fonte e estilos extras

└── README.md => Este arquivo


🛠️ Tecnologias

HTML5, Tailwind CSS (via CDN), Fonte Inter, SVG

▶️ Como executar


Mantenha index.html e styles.css na mesma pasta.
Dê um duplo clique em index.html.

⚠️ O Tailwind é carregado via CDN — é preciso internet na primeira abertura.

📱 Responsividade

Layout adaptável: no desktop os cards ficam lado a lado; no mobile, empilham verticalmente.
